# FizzBuzz Assignment

## Description
This program prints the numbers from 1 to 100 to the console with special rules:

- If a number is divisible by 3, it prints "Fizz"
- If a number is divisible by 5, it prints "Buzz"
- If a number is divisible by both 3 and 5, it prints "FizzBuzz"
- Otherwise, it prints the number itself

## Files Included
- FizzBuzz.js – JavaScript program that runs the FizzBuzz logic
- README.md – Documentation for the assignment
