# Fahrenheit to Celsius Converter 

## Description:
This JavaScript program converts 99 degrees Fahrenheit to Celsius.

## Variables:
- f = stores the Fahrenheit value (99)
- c = stores the Celsius result

## Formula Used
Celsius = (Fahrenheit - 32) * 5 / 9

## Output
Fahrenheit: 99
Celsius: 37.22222222222222
